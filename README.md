# Zen Site

A calm, minimal wellness landing page built with Next.js and Tailwind CSS.

## Features

- Soft sage and stone color palette
- Interactive breathing exercise
- Responsive, accessible layout
- Serif + sans typography pairing (Cormorant Garamond + DM Sans)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to [Vercel](https://vercel.com) or any Next.js-compatible host.

```bash
npm run build
```
